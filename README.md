# Helptopia
Guia para iniciantes do jogo Heartopia!

## Como executar

### Pré-requisitos
- Node.js instalado (versão 14 ou superior recomendada)

### Instalação
1. Clone o repositório:
   ```bash
   git clone https://github.com/Felipssv/Helptopia.git
   cd Helptopia
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

### Executando o projeto
- Para executar em modo produção:
  ```bash
  npm start
  ```

- Para executar em modo desenvolvimento (com nodemon para recarregamento automático):
  ```bash
  npm run dev
  ```

O aplicativo será executado em `http://localhost:3000`. Abra seu navegador e acesse essa URL para visualizar o guia.

## Estrutura do projeto
- `index.js`: Arquivo principal do servidor Express
- `views/`: Templates EJS para as páginas
- `public/`: Arquivos estáticos (CSS, imagens)
- `package.json`: Dependências e scripts do projeto
